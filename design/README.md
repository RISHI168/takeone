# TakeOne Design System

This folder contains **38 interactive HTML prototypes** across two visual themes, covering every core screen of the TakeOne platform.

## How to View

Open any `.html` file directly in your browser — no build step or server required. Each file is self-contained with inline Tailwind CSS.

## Themes

| Theme | Folder | Screens | Description |
|-------|--------|---------|-------------|
| Cinematic Dark | `cinematic-dark/` | 19 | Premium dark aesthetic with accent colors. Most complete set. |
| Ivory Light | `ivory-light/` | 19 | Clean light theme for professional contexts. |
